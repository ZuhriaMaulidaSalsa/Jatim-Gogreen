Spesifikasi Ringkasan Kesesuaian
Section Ada 5 section yaitu header, menu, content, sidebar, dan footer
Jumlah Halaman Ada 4 Halaman, yaitu :
  • Homepage berisi segala informasi dan link yang berkaitan dengan website yang sesuai dengan tema yang telah ditentukan.
  • Form berisi data inputan relawan
  • Evaluation berisi ringkasan untuk menjelaskan kelengkapan dan kesesuaian dari website yang dibuat terhadap spesifikasi tugas.
  • About Us berisi nama mata kuliah, kelas, nama kelompok serta identitas mahasiswa, dan tabel untuk menjelaskan pembagian pengerjaan. Dan juga berisi URL dari gambar yang digunakan website yang tersedia online.
Nama File untuk Halaman Pertama Nama file halaman utama sesuai dengan aturan yaitu index.html
Penggunaan Elemen HTML form Lengkap : textbox, dropdown menu, check box, radio button, text area, dan submit button
Jenis Validasi Form Validasi form lengkap : Masukan wajib, format numerik, format alfabet, format alfanumerik, format email, masukan dengan panjang digit dan panjang karakter tertentu, dan masukan kesesuaian tanggal
Pesan Error Validasi Form Untuk pesan eror validasi form kami menggunakan javascript(innerHTML) tidak menggunakan kotak dialog alert
